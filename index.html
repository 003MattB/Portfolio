Featured Projects

Documentation For Genesis Interface Library

I wrote this library (in Perl) to interface with Genesis Cam Software scripting mechanism. I began this project in 2017 after I had been writing automation scripts for a while, before starting school. I realized that many scripts shared a lot of functionality, namely you need to get information about the job like: attributes and the structure. After finding a bug in one script, and needing to fix that bug in multiple scripts I decided I need to consolidate the common functionalities so this would not happen again. Thus came the "Job" module.

This principle is pretty obvious to me now; Don't Repeat Yourself or DRY for short, but cut me some slack it was before I started my Bachelors.

Over the years I've been adding more and more modules to the library. In order to keep all the documentation in a centralized location, I made this. It's an easy to navigate, online hub of information.

Schedule Builder

The idea for this came to me when I was trying to build a class schedule with the University's Schedule Builder, but it's 15 schedule limit would not come up with something that would work for my work schedule. I wanted to be able to see all available sections at once for all the classes I wanted to take. What I ended up doing was write all my desired classes, with every section, on note cards and lay them out on a week schedule. I was able to visualize a working schedule, but it was a ton of work.

So, I came up with this. It pulls information from umn-asr/courses API and puts all available sections into a schedule. You can then manually remove cards form the schedule by clicking the x in the upper right corner. It will also keep track of the total credits which are in the schedule.

Perfect Scale

Created for Pro-tech Interconnect Solutions, Perfect Scale is a command line tool which processes a quadrant-view x-ray image of a circuit boards corner drilling registration targets and calculates a "perfect scale" and zero-correction required for the drills to align with the PCB image.

In this project, I learned how to do some basic image processing using the OpenCV library for Python.
